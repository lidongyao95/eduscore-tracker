EduScore Tracker Architecture

高校教学前测后测评测平台 — Flask + SQLite + Bootstrap 5 + ECharts

Frontend Stack

  • • Bootstrap 5.3 — BootCDN 国内镜像
  • • ECharts 5.5 — 成长轨迹可视化
  • • Bootstrap Icons 1.11 — UI 图标
  • • Jinja2 SSR — 服务端模板渲染
  • • 16 templates — 教师 + 学生 + 共享

Backend Core

  • • Python 3.12 + Flask 3.0
  • • 3 Blueprints: Admin · Student · Auth
  • • Flask-Login: Session + Role Guards
  • • Flask-WTF: CSRF Protection
  • • Flask-Caching: 60s TTL SimpleCache

Data & Analytics

  • • SQLite — 零配置单文件数据库
  • • SQLAlchemy ORM — 8 tables, lazy='select'
  • • gain.py — score_rate · Hake g · class summary
  • • growth.py — growth context · risk_level · recommendations
  • • 学情分析: 风险等级 · 薄弱单元 · 教学建议
