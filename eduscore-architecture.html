EduScore Tracker — System Architecture

高校教学前测后测平台 · Python Flask · SQLite · Bootstrap 5 · ECharts

Core Business Logic

  • • Pre-test → Post-test gain analysis (Hake's g)
  • • Multi-attempt submissions with auto-grading
  • • Learning objective-based score breakdown
  • • Short-answer manual grading by teacher
  • • ECharts growth trajectory visualization

Security & Data Isolation

  • • werkzeug.security scrypt password hashing
  • • Flask-Login session management
  • • Flask-WTF CSRF protection on all forms
  • • Role-based access: student vs teacher
  • • Teacher ↔ own classes / Student ↔ own submissions

Caching & Performance

  • • SimpleCache (in-memory dict), zero external deps
  • • build_student_growth_context → @cache.memoize(60)
  • • class_gain_summary → @cache.memoize(60)
  • • Write-through invalidation: delete_memoized on submit/publish/edit
  • • SQLite with lazy='select' relationship loading
